M.E.R.N. Stack Components

Core Script: javaScript

The Core Programming language which is the backbone of this Full Stack Application

Web Framework: Express.js

The Express Framework written in javaScript serves as the structure for which your application will speak to the server

Programming Language: Node.js

The Backend Programming Language will be structured using Node and its CLIs NPM & NPX

Database: MongoDB

The "M" in MERN Stack stands for MongoDB the non-sql database structure that will serve as the bucket to store our data

JS Framework: React.js

To build our application with slick and beautiful User Interface & User Experirence designs we will use the React.js

Key Libraries: Materials & Bootstrap

To simplify and speed up our construction we will build from ready to use cut-copy-past libraries

Deployment: Cloudinary & Heroku

Deploying our application and storing our media will be made much easier using the Cloudinary CDN & Heroku platforms

Testing Tools: Postman & Netlify

Testing never gets easier than when using Postman & Netlify
